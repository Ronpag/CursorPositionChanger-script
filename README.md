<h1 align=center> Cursor Position Changer </h1>
<h3>This is a simple script for trading on the market. The program is optimized for a 1920x1080 screen (I think). </h3>

<p>Functions:</p>
<ul>
  <li>Tab - Close the market tab</li>
  <li>Q - Select “Create Buy Order”</li>
  <li>W - Add one unit to the price</li>
  <li>E - Click “Order”</li>
  <li>D - Click “Agree”</li>
  <li>Spacebar - When updating an order, it quickly sets the amount to 5 and closes it.</li>
  <li>R - When updating an order, it quickly sets the amount to 7 and closes it.</li>
  <li>F - When updating an order, it quickly sets the amount to 3 and closes it.</li>
</ul>
